# Blinkit Sales Analysis – SQL

## Overview

This project analyzes Blinkit retail sales data using SQL to extract business insights related to product performance, outlet characteristics, and sales distribution. The analysis focuses on key performance metrics such as total sales, average sales, item count, and customer ratings.

## Objectives

* Analyze overall sales performance of Blinkit outlets
* Identify sales contribution by outlet size, location, and type
* Evaluate product category performance
* Generate KPI metrics to support business insights

## Tools Used

* SQL Server Management Studio (SSMS)
* SQL

## Key Analysis Performed

* Calculated total sales, average sales, and average ratings
* Analyzed sales by item type and fat content
* Evaluated outlet performance by location and establishment year
* Used **GROUP BY**, **PIVOT**, and **window functions (OVER())** to derive insights
* Calculated sales contribution percentages across outlet categories

## Key Insights

* Medium-sized outlets generated the highest sales contribution
* Certain item categories showed significantly higher average sales
* Outlet location and type influenced overall revenue distribution

## Dataset

The dataset contains Blinkit retail data including:

* Item Type
* Item Fat Content
* Outlet Size
* Outlet Location Type
* Outlet Establishment Year
* Total Sales
* Rating
